# Pedra, Papel e Tesoura

Um jogo simples de pedra, papel e tesoura contra o computador programado em python.

## Como jogar
1. execute o arquivo ´pedra-papel-tesoura.py`
2. escolha pedra, papel ou tesoura
3. tente vencer o computador

## Exemplo:
Escolha pedra, papel ou tesoura: tesoura  
Computador escolheu: pedra  
Você perdeu!
